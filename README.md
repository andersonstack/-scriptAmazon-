# Amazon Scraper (JavaScript)

A lightweight web scraper that extracts product listings from Amazon search results, built with:
- **Backend**: Bun + Express
- **Frontend**: Vite + Vanilla JavaScript
- **Scraping**: Axios + JSDOM
